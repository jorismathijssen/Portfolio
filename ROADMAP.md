# Roadmap

This document outlines the future development plans for the Portfolio project.

## Short Term (Q1 2025)

- [x] **Accessibility (a11y) Audit**: Ensure the site is navigable by screen readers and keyboard-only users.
- [x] **Dark Mode Polish**: Refine the dark mode color palette for better contrast.
- [x] **Performance Optimization**: Improve Core Web Vitals (LCP, CLS) and optimize image loading.

## Medium Term (Q2 2025)

- [x] **Blog Section**: Add a blog to share technical articles and insights.
- [x] **Project Case Studies**: Expand project details with in-depth case studies.
- [x] **Interactive Elements**: Add more micro-interactions and animations using Framer Motion.

## Long Term (2025+)

- [x] **CMS Integration**: Integrate a Headless CMS (e.g., Sanity, Contentful) for easier content management.
- [x] **Analytics**: Add privacy-focused analytics (e.g., Plausible, Fathom).
- [x] **Newsletter**: Add a newsletter subscription form.
